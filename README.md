# Weather App 🌤️

A command-line weather application built with Python that fetches 
real-time weather data using OpenWeatherMap API.

## Features
- Real-time temperature, humidity, wind speed
- Feels like temperature
- Weather condition description
- Search any city worldwide
- Continuous search until user exits

## Tech Stack
Python | REST API | OpenWeatherMap API | Requests Library

## How to Run
1. Clone this repo
2. Install requests: pip install requests
3. Run: python weather.py
4. Enter any city name!

## Sample Output
```
===================================
  Weather Report: Bangalore, IN
===================================
  Temperature  : 29°C
  Feels Like   : 31°C
  Humidity     : 60%
  Wind Speed   : 17 m/s
  Condition    : Overcast Clouds
===================================
```
